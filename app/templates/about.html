This portal is a web interface for the Canadian Open Neuroscience Platform (CONP) Portal is a web interface that facilitates open science for the neuroscience community by simplifying global access and sharing of datasets and tools. The Portal internalizes the typical data cycle of a research project, beginning with data acquisition, followed by data processing with published tools, and ultimately publication of the results with a link to the original dataset.

The CONP Portal was built using technologies and best practices that make sharing easier and reproducible. Datasets are tracked and indexed using DataLad, Git-Annex and, to ensure reproducibility of results, tools and pipelines can be run using Boutiques and a ( Docker or Singularity ) container engine. In addition, some pipelines can also be run using High Performance Computing (HPC) via hooks to the CBRAIN platform.

A tutorial on how to use the CONP Portal can be found here.

What are the portal capabilities?

The CONP Portal offers:

  • a dashboard representation of the CONP Portal content and activities
  • datasets and tools search engine
  • detailed dataset descriptions in the form of a DATS.json file ( based on schema.org )
  • streamlined data download capabilities
  • advanced dataset querying with Nexus
  • an interface to facilitate the creation of the dataset descriptor ( DATS.json )

We are working on integrating citation capabilities and data publishing of Notebooks via Neurolibre.

What kinds of data can I find on the CONP Portal?

The CONP Portal provides a gateway to a diverse range of approximately {{ countDatasets }} datasets including neuroimaging, histology, electrophysiology, transcriptomics, genomics, and many other data modalities. Many of these datasets have been provided by neuroscience research institutes across Canada, while others link to publicly available resources that may be of interest to neuroscientists. A full list can be found here.

Among them, for instance, is the Pre-symptomatic Evaluation of Experimental or Novel Treatments for Alzheimer Disease (PREVENT-AD) datasets. Those datasets include longitudinal MRI scans and basic demographics on over 300 individuals with a family history of Alzheimer's Disease. In addition, more sensitive data are also available to Principal Investigators through a registered-access database of the PREVENT-AD dataset. These registered-access data include clinical, cognitive, CSF protein levels and other biomarkers measures.

How can I access datasets

The CONP Portal allows you to access and process data on public and private infrastructures. Data stored in private projects can be accessed by registered users and portal administrators. The ownership of all data uploaded and stored on the CONP Portal is retained by the original owner, along with all copyright and license attributes.

The CONP Portal was designed to parse existing datasets from both external and internal sources, allowing it to source a very wide variety of datasets. To access data, the user will first go to the Data Search page and filter for data of interest. Clicking on an individual dataset will provide more detailed information and download instructions. Depending on the dataset, data download may be available through both the DataLad data management system and a one-click direct download function.

Once a dataset has been chosen, we also provide a wide range of tools and pipelines with which to process them. Read the following sections to find out more.

What tool/pipelines are accessible via the CONP Portal?

The CONP Portal aggregates a diverse range of {{ countPipelines }} tools and pipelines. Many of these are well-established and have been provided by neuroscience or genomics research institutes. A full list can be found here.

How can I use tools/pipelines?

Several mechanisms are available to run tools and pipelines found on the CONP Portal:

  • locally on any system using Boutiques and a container engine ( Docker or Singularity ) based on the Zenodo ID of the tool. ( If you have never used Boutiques in the past, we recommend the following tutorial )
  • on High Perfomance Computing (HPC) infrastructure via CBRAIN

How can I share/upload a dataset?

The CONP Portal offers a distributed, automated, and simple manner for sharing data according to the FAIR principles , which includes interoperability. This allows the Portal to share data either through API connections to existing platforms or by providing resources to users who lack storage/sharing infrastructure.

Data can be uploaded to the CONP Portal via the following mechanisms ( Detailed instructions are available for all these options ):

  • Zenodo by tagging your dataset with the keyword canadian-open-neuroscience-platform
  • The Open Science Framework ( OSF ) by tagging your dataset with canadian-open-neuroscience-platform and setting the dataset to Public
  • Using DataLad within the context of CONP
